## Core

Emotion = evidence, not label.

Prose supplies body / voice / action / object / environment evidence.

Reader classifies.

Evidence already shows the state → delete explanation.

## Beat

Beat = one emotional moment.

Beat ends when register / action / topic / distance / power balance shifts.

Same beat → do not repeat same evidence channel.

New beat → new channel combination.

## Show-Then-Tell Cut

Action + explanation → delete explanation.

Bad:

- Description: 코끝에서 바람이 새어나왔다. 웃음이라기엔 너무 작았다.

Good:

- Description: 코끝에서 바람이 새어나왔다.

## Evidence Channels

Available families:

- muscle/posture: shoulder / spine / neck / jaw / freeze
- breath/voice: held inhale / cracked voice / trailing words / swallow
- gaze/expression: landing point / avoidance / blink / brow / mouth tension
- hands/fingers: grip / fidget / clench / release / object handling
- rhythm: speed / delay / repetition / mechanical action
- environment: temperature / sound distance / light / air / object stillness

Same body reaction max once per scene unless function changes.

Same feeling returns → reroute evidence.

Same body part 3+ times for same emotional function = forbidden.

## Compound Emotion

Mixed emotion ≠ single label.

Use two visible impulses that do not agree.

Good:

- Description: 입꼬리가 올라갔다. 그녀의 손가락은 컵 손잡이 주변을 맴돌고 있었다.

Depth = contradiction in action, not darker vocabulary.

## Dialogue Gap

Dialogue may oppose body / substitute for feeling / evade topic.

e.g. "늦었잖아." / "밥은 먹었어?" / "별거 아니야."

Gap carries emotion. Do not explain gap after showing it.

## Inner Content

Inner content may state thought caused by emotion, not emotion label.

Good:

- Italic: *괜히 말했나.*

Bad:

- Italic: *나는 후회했다.*

Inner content = brief + character-voiced + immediate.

## Environment

Environment may reflect pressure through physical facts only.

e.g. cup cooling / fluorescent hum / rain on glass / chair scrape

Atmosphere label = forbidden.

Character emotion may color object/person description through personification. This is character voice, not atmosphere label.

Environment projection max 1 per beat.
